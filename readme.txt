== Description ==

The Create Child Theme plugin lets you create a new child theme immediately, within the WordPress admin interface. 


== Installation ==

* Upload the Create Child Theme plugin to the `/wp-content/plugins/` directory
* Activate the Create Child Theme plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==
 
= Where is menu of Create Child Theme plugin? =
* Once plugin installed and activated, you can find it in WordPress's Tools - Create Child Theme.

= How to create child theme in plugin menu? =
* First select a parent theme. 
* Customise options such as title and description, etc. Help text is available on right of each field. 
* You have the option to include GPL License Terms. 
* Click Create Child Theme. 
* A child theme thumbnail will be created automatically. 
* Upon success you will find your new child theme under WordPress's Appearance - Themes, where it can be Activated.

= What files are there in newly created child theme? =
* Newly created child theme has style.css file with code written in it and readymade functions.php file.
